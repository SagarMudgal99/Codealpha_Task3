# 🚗 Car Price Prediction with Machine Learning

This project develops a Machine Learning regression model to predict car prices based on various features such as brand, horsepower, mileage, fuel type, and other vehicle specifications. The project covers data preprocessing, feature engineering, model training, evaluation, and prediction.

## 🚀 Features

* Data cleaning and preprocessing.
* Feature engineering and selection.
* Train a regression model for car price prediction.
* Evaluate model performance using regression metrics.
* Visualize relationships between features and car prices.

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## 📊 Dataset

The dataset contains car-related attributes including brand, mileage, horsepower, fuel type, and other specifications used to predict vehicle prices.

## 🎯 Objective

To build a machine learning model capable of accurately predicting car prices and demonstrate the practical application of regression techniques.
